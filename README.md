# tox-docker-typo3

playgound for TYPO3 for docker using docker-compose and docker-sync


## Usage

### macOS

You have to install docker-sync.io ...

#### start

    docker-sync-stack start

#### stop

At first CMD+C ...

    docker-sync-stack clean

### linux

currently untested ...

#### start

    docker-compose up -d

#### stop

    docker-compose down
